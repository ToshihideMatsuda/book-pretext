# CLAUDE.md

## プロジェクト概要

- このリポジトリは、日本文学テキスト向けの小さな Vite + TypeScript 製ビューワです。
- アプリ本体のロジックは主に `src/main.ts` にあります。
- `index.html` に UI のシェルと大半のスタイルがあります。
- `books/` には実行時に読み込まれるテキストファイルがあります。

## 作業方針

- 要求を満たすための最小限の変更を優先してください。
- 明示的な依頼がない限り、既存の読書体験や操作感は維持してください。
- 正しさのために必要でない限り、`src/main.ts` の大きなリファクタリングは避けてください。
- `dist/` や `node_modules/` のような生成物や依存ディレクトリは編集しないでください。

## 実装メモ

- 書籍データは `import.meta.glob('../books/**/*.txt', ...)` で収集しています。
- 検索、目次、ページ移動、タッチ操作はすべて `src/main.ts` で連携しています。
- タップ、スワイプ、クリック、オーバーレイ周りの変更では、デスクトップとモバイルのイベント競合を確認してください。
- 新しい補助関数は、必要以上に抽象化せず、ローカルで素直な実装を優先してください。

## 検証

- コード変更後は `npm run build` を実行してください。
- UI 挙動を変更した場合は、少なくとも以下を手動確認してください。
  - ページ移動
  - タップとスワイプの挙動
  - 検索パネルの開閉と検索結果移動
  - 目次の開閉と書籍切り替え

## 編集の好み

- 過度に凝った抽象化より、短く読みやすいコードと明確な命名を優先してください。
- デザイン変更の依頼がない限り、既存の見た目やトーンを保ってください。
- コメントは最小限にとどめ、コードだけでは意図が追いづらい箇所にだけ付けてください。

## 返答時の期待

- 変更したファイルを簡潔に要約してください。
- `npm run build` を実行した場合は、その結果を報告してください。
- 未確認の点がある場合は、その旨を明示してください。
